# Data Sales Dashboard (Excel)

Dashboard de vendas desenvolvido em **Microsoft Excel** como parte de um desafio de análise de dados.
O objetivo do projeto é transformar dados brutos em **informações visuais claras**, permitindo análise de desempenho e apoio à tomada de decisão baseada em dados.

---

# 📊 Objetivo

Criar um **dashboard de vendas interativo** utilizando Excel para visualizar métricas importantes relacionadas a assinaturas de serviços.

O projeto busca demonstrar habilidades em:

* Organização e estruturação de dados
* Construção de métricas de negócio
* Visualização de dados
* Criação de dashboards analíticos
* Uso do Excel para análise de dados

# 📂 Estrutura do Arquivo Excel

O arquivo `xbox_dashboard_vendas.xlsx` está organizado em diferentes abas para separar **dados, cálculos e visualização**.

### Dashboard

A aba principal do projeto, contendo o painel visual de análise.

Apresenta indicadores de desempenho como:

* Receita total
* Número de assinantes
* Distribuição por plano
* Comparação entre tipos de assinatura
* Indicadores visuais para análise rápida

O objetivo é permitir que o usuário compreenda rapidamente o desempenho das vendas.

---

### Bases

Contém a **base de dados utilizada no dashboard**.

Entre as informações disponíveis estão:

* Identificação da assinatura
* Tipo de plano
* Valores de assinatura
* Cupons aplicados
* Valor final pago pelo cliente

Essa aba representa os **dados brutos utilizados para gerar as métricas**.

---

### Cálculos

Responsável por gerar as **métricas agregadas** utilizadas no dashboard.

Exemplos de cálculos realizados:

* Receita total por plano
* Quantidade de assinantes
* Receita média por assinante
* Percentual de participação por plano

Esses dados alimentam diretamente os gráficos e indicadores do painel.

---

### Assets

Contém elementos auxiliares utilizados na construção do dashboard, como:

* Paleta de cores
* Elementos visuais
* Padronização de layout

Essa aba ajuda a manter **consistência visual no painel**.

# ⚙️ Como Utilizar o Projeto

1. Clone o repositório ou baixe o arquivo xlsx:

```
git clone https://github.com/kalkehcoisa/dio-data-dashboard.git
```
Ou:
[xbox_dashboard_vendas.xlsx](./xbox_dashboard_vendas.xlsx)

2. Abra o arquivo no Microsoft Excel:

[xbox_dashboard_vendas.xlsx](./xbox_dashboard_vendas.xlsx)

3. Acesse a aba **Dashboard** para visualizar o painel.

4. Caso queira entender os dados e cálculos:

* Consulte a aba **Bases**
* Consulte a aba **Cálculos**
